When things go wrong
Troubleshooting
ProblemFix
PWA install icon not showing in Chrome Hard refresh: Cmd+Shift+R. Wait 10 seconds on the page. Check Chrome ‚ãÆ menu for install option.
App not updating after GitHub change Wait 60 seconds after committing. Then hard refresh: Cmd+Shift+R. Or close and reopen the app.
API call fails / spinning forever Wait for the 180s cooldown. Check your internet connection. If it persists, reload the page ‚Äî your editor data is saved in localStorage.
429 error Too many API requests. Engine auto-retries after 90s. Just wait ‚Äî don't click again.
Lost my work after reload Click üìÅ Issue Archive ‚Äî if you saved before reloading it'll be there. Otherwise check Downloads for any exported HTML files and use Import Issue.
Preview shows raw HTML code Click the Preview button in the mode toggle (top right of main area) to switch to rendered view.
GitHub Pages not updating Go to repo ‚Üí Settings ‚Üí Pages ‚Äî check for a yellow building indicator. Wait for green checkmark, then hard refresh.
Can't find the app on Mac Cmd+Space ‚Üí type FGWW or Food Game. Or open Launchpad (rocket icon in Dock) and look there.
Substack not accepting HTML Look for the </> HTML toggle in Substack's editor toolbar. Some browsers need you to use the Edit menu ‚Üí HTML Source.
